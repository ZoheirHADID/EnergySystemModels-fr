Introduction à la Production Photovoltaïque
===========================================

Le module PV d'EnergySystemModels permet de simuler la production électrique de systèmes photovoltaïques en utilisant la bibliothèque pvlib.

Fonctionnalités
---------------

* Simulation de production photovoltaïque
* Calcul de l'irradiance solaire (GHI, DNI, DHI)
* Prise en compte de l'orientation et de l'inclinaison
* Intégration de données météorologiques
* Calcul des pertes et rendements
* Visualisation de la production annuelle

Concepts clés
-------------

Irradiance solaire
~~~~~~~~~~~~~~~~~~

* **GHI** (Global Horizontal Irradiance) : Irradiance sur plan horizontal
* **DNI** (Direct Normal Irradiance) : Irradiance directe normale
* **DHI** (Diffuse Horizontal Irradiance) : Irradiance diffuse

Paramètres d'installation
~~~~~~~~~~~~~~~~~~~~~~~~~~

* **Puissance crête (kWc)** : Puissance nominale du système
* **Azimut** : Orientation (0° = Sud, 90° = Ouest, -90° = Est)
* **Inclinaison (tilt)** : Angle par rapport à l'horizontale
* **Pertes système** : Câblage, onduleur, salissure, ombrage

Applications
------------

* Dimensionnement d'installations PV
* Études de faisabilité
* Optimisation de l'orientation/inclinaison
* Prévision de production
* Autoconsommation solaire
